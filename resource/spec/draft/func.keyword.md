# Keyword
It's commonly known as an entry in index page.
It will query all of the pages contain the specified keyword
```gularen
% keyword "Mythology"
```
```html
<div class="keyword">
    <b>Mythology</b>
    <span>
        <a href="#page-43">43</a>
        <a href="#page-59">59</a>
    </span>
</div>
```
